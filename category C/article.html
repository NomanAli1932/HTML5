Understanding Semantic HTML

Why Semantics Matter

Semantic HTML helps improve accessibility, search engine optimization, and code readability.
As Tim Berners-Lee once said, Data is a precious thing and will last longer than the systems themselves.

Real-World Impact

Using the correct tags, like <article> or <section>, makes your
content more meaningful. Always try to write code that both machines
and humans can understand. The project is due on Nevomber 05, 2025, so plan accordingly.

The blockquote element represents content that is quoted from another source.

Sample Code

<section>
  <h2>About Us</h2>
  <p>We build semantic websites.</p>
  <h3>contact No<\h3>
  <p> 0345946793 <p>
</section>
